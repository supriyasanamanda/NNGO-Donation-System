# NSS NGO Donation System

## Setup Instructions
1. **Backend:**
   - `cd backend`
   - `npm install`
   - Create a `.env` file (see below)
   - `npm start`
2. **Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm start`
